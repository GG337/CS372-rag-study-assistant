# Setup Instructions

## Requirements

- Python 3.10 or higher
- pip

## Installation

Install dependencies:
    ```python -m pip install -r requirements.txt```

## Running the Project

Run the test script: 
    ```python src/test2.py```

## Notes

- The first run may download pretrained models (this can take a few minutes)
- No GPU is required; everything runs on CPU
- Make sure you are in the project root directory when running commands
